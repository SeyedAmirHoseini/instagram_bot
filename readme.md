# Instagram Movie Bot (ربات اینستاگرام دانلود فیلم و سریال)

**نسخه نهایی - دسامبر ۲۰۲۵**

این ربات برای اکانت اینستاگرام شما طراحی شده تا کاربران با کامنت گذاشتن یک کد ۴ رقمی زیر پست‌ها، لینک دانلود فیلم/سریال/انیمیشن را به صورت خودکار در دایرکت دریافت کنند.

ادمین (شما) از طریق **Saved Messages** (پیام به خودتان) پنل مدیریت را کنترل می‌کند.

## ویژگی‌ها
- ثبت محتوا با دستور ساده: `{ثبت {لینک اپلود} - {نام فیلم`
- پنل ادمین کامل با دستور `پنل`
- تنظیم فاصله چک کامنت‌ها، تأخیر دایرکت‌ها و تعداد پست تحت نظارت
- ارسال خودکار لینک دانلود در دایرکت به کامنت‌گذاران
- جلوگیری از پردازش تکراری کامنت‌ها (با استفاده از دیتابیس)
- فقط Saved Messages (پیام به خودتان) را چک می‌کند → بهینه و کم‌خطر
- متن خروجی ثبت به صورت بلوک کد → کپی آسان با یک لمس

## پیش‌نیازها
- Python 3.10+
- کتابخانه‌ها:
  pip install -r requirement.txt


## ساختار پروژه
instagram-bot/
├── main.py                 # نقطه شروع ربات
├── config.py               # تنظیمات ثابت (نام کاربری، ربات تلگرام و ...)
├── .env                    # اطلاعات حساس
├── bot_data.db             # دیتابیس SQLite (فیلم‌ها، تنظیمات، پردازش‌شده‌ها)
├── session.json            # جلسه لاگین اینستاگرام (تولید می‌شود)
└── utils/
    ├── bot.py              # کلاس اصلی ربات
    └── link_processor.py   # پردازش لینک تلگرام و تولید عدد رندوم

## نحوه راه‌اندازی

فایل .env را بسازید و اطلاعات را وارد کنید:
```
INSTA_USERNAME=your_username
INSTA_PASSWORD=your_password
# INSTA_SESSIONID=...   (اختیاری - اگر دارید)
TELEGRAM_BOT_USERNAME=MoviesssBot_bot   # بدون @
ADMIN_ID=123456789012   # user_id اینستاگرام خودتان
```

# ربات را اجرا کنید:
```
py main.py
```
